JinjaDCI: a Java semantics with dynamic class initialization
Susannah Mansky

We extend Jinja to include static fields, methods, and instructions, and dynamic
class initialization, based on the Java SE 8 specification. This includes
extension of definitions and proofs. This work is partially described in Mansky
and Gunter's paper at CPP 2019 and Mansky's doctoral thesis (UIUC, 2020).

https://www.isa-afp.org/entries/JinjaDCI.html
