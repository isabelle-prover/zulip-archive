Topic: UI Scaling

Mario Xerxes Castelán Castro (Nov 16 2025 at 23:00):

I downloaded the .tar.gz of Isabelle for GNU/Linux. The jEdit-based editor seems to not obey my system-wide UI scaling setting so the UI is way too small. How can I make it bigger?

Mario Xerxes Castelán Castro (Nov 17 2025 at 00:54):

I added this in the file Isabelle.options:

-Dsun.java2d.uiScale.enabled=true
-Dsun.java2d.uiScale=2

Unfortunately it only works with integer scaling factors.
