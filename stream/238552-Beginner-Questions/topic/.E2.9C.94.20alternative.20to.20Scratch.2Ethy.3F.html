On double clicking the Isabelle icon on my desktop Isabelle/jEdit will open the file Scratch.thy in my ISABELLE_HOME_USER directory or with an empty buffer named Scratch.thy in case there is no such file. Lukas Stevens (Nov 23 2021 at 10:44):

I am not too familiar with Windows but I think you should be able to change the command that is executed when inspecting the settings of the icon.

Lukas Stevens (Nov 23 2021 at 10:45):

If you change the command to isabelle jedit Test.thy, then it will open Test.thy.

Edmund Dikow (Nov 23 2021 at 12:59):

Thank you, that works. Two additional observations should be noted

  1. On Windows machines the link created during an Isabelle installation is simply to Isabelle2021.exe (not isabelle jedit).
    e.g. if my Isabelle installations reside in C:\Users\UserName\Isabelle and my theory files reside in E:\theoryfiles,
    I can set the target of the Isabelle2021 icon on the desktop to
    C:\Users\UserName\Isabelle\Isabelle2021\Isabelle2021.exe E:\theoryfiles\myTheory.thy

  2. I have to restart my computer for a change in the link to become active.
