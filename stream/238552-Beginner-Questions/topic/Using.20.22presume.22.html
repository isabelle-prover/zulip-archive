Balazs Toth (Jan 28 2025 at 15:10):

It's actually the first time that I hear of presume, but I think I figured out how to use it. This should be the typical pattern on how to use it:

lemma
  assumes "A" "A ⟹ B"
  shows "B"
proof-
  presume A: "A"

  then show ?thesis
    using assms(2)
    by simp
next
  show "A"
    using assms(1)
    by simp
qed
