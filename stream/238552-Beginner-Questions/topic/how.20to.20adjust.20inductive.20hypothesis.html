From your description, this sounds like you are trying to prove:

lemma
  assumes ‹list_all2 P x y› ‹Q [] [] []›
    ‹⋀a b x y c d. Q [x] a b ⟹ Q y c d ⟹ Q (x#y) c d›
  assumes ‹⋀x y. P x y ⟹ ∃a b. Q [x] a b›
  shows ‹∃a b. Q x a b›
  using assms
  by (induction rule: list_all2_induct)
    fastforce+
