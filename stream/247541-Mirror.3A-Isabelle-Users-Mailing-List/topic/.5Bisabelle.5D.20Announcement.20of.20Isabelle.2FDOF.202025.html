Dear all,

we happily announce the recent version of Isabelle/DOF (2025), a
Document Ontology Framework (DOF), on top of Isabelle/HOL,
allowing to annotate text elements in formal developments with
structured, typed meta-information which can be defined by developers
according to their purposes in an Ontology. Generally speaking,
DOF-generated meta information can be used for document generation,
semantic queries, and tool interaction.

We use DOF to write our papers and technical documentation systematically;
the ZENODO page https://zenodo.org/records/15274072 <https://zenodo.org/records/15274072> gives examples and
the DOF related publication bibliography.

DOF comes in two parts, both of which were best used if registrated as
Isabelle component.

  • The “Core” which is available as AFP component.
    it suffices to be included if sessions use DOF.

  • The Add-On Package, which provides examples, more non-standard
    ontologies and, last but not least, an incremental document generator
    (granularity: thy-files).

More details, installation hints nd the reference manual can be received from
the ZENODO page.

Have fun !

Achim Brucker and Burkhart Wolff
