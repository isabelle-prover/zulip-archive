Dear Isabelle/Haskell users,

I am happy to announce the immediate availability of The Isabelle Cabal

  • a Cabal plugin providing support for Isabelle code generation to
    Haskell as part of the usual Cabal build process.
    This allows you to build Haskell applications which depend on code
    extracted from Isabelle theories without messing around with multiple
    build systems or manual extraction steps.

The package, including instructions for use, is available from:
https://github.com/fkj/isabelle-cabal

Additionally, a demo package showcasing basic use of the plugin is
available from:
https://github.com/fkj/isabelle-cabal-demo

The Isabelle Cabal is still early in development, but is stable enough
for general use.
If you encounter any problems while attempting to use it, please let me
know either by opening an issue on the GitHub repository or by emailing me.
If you have any questions or ideas, please also feel free to contact me.

Cheers,
Frederik
