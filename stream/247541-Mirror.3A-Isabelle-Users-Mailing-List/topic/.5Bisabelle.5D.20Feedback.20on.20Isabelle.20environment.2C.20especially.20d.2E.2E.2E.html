Hello

I just integrated Isabelle into my workflow. Let me use this opportunity
give a little feedback (I am using Isabelle 2025 under Windows):

* Isabelle does not start if other programs opened by Isabelle are
still running, e.g. manuals.
* How do I call Isabelle's cygwin from jEdit (or somehow more
directly)?

NO BIBLATEX-SUPPORT

*

No command biblatex_file. As far as I can see, the essential definitions
for this are in Pure/General/bibtex.scala. IF THERE IS INTEREST, I WOULD
BE HAPPY TO CREATE A BIBLATEX.SCALA.
*

It took me a working day and AI to figure out how to finally include
BibLaTeX. Ideally, once a biblatex_file command is called, the build
file should be automatically adjusted so that biber is called.

DOCUMENT PANEL

*

gives no error feedback when sessions are not loaded in the ROOT file
(in contrast to the command line).
*

doesn't create a folder with files, like the command line does. Why this
difference? Shouldn't there be an option to switch between both
behaviours?

ROOT AND ROOT FILES

*

There seems to be no convenient way to change the root file by a command
like document_root = root_luckhardt.tex.
*

When end-of-line characters in BOOT files are messed up, only a general
"Corrupted character" message is coming. Ideally, there would just a
button to be hit to fix the problem.
