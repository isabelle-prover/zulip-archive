Dear Isabelle users,

we are still approx. 5 weeks before the official start of the Isabelle2022
process: final lift-off is anticipated for late October 2022.

The cumulative blog entry is here:
https://isabelle-dev.sketis.net/phame/post/view/58/release_candidates_for_isabelle2022

Presently there is only https://isabelle.sketis.net/website-Isabelle2022-RC0
to demonstrate that it is possible to make an interactive prover distribution
at almost 2 GB directory size. It contains full Java 17, Scala 3, and now also
VSCode/Electron/Node.js. As a corollary we have a complete Chromium browser
engine bundled with Isabelle.

I am curious to see what we can do with that in the near future. My virtual
talk this Thursday will only speculate about a few possibilities.

Makarius
