Dear Isabelle users,

this is the start of the official release process for Isabelle2022 (October
2022). We have approx. 6 weeks left to sort out remaining problems; afterwards
the release will become final and remain unchanged (until the next release
after it).

The release process is continuously documented on the blog entry
https://isabelle-dev.sketis.net/phame/post/view/58/release_candidates_for_isabelle2022

The current release candidate is
https://isabelle.sketis.net/website-Isabelle2022-RC1

A corresponding version of the Archive of Formal Proofs is
https://isabelle.sketis.net/repos/afp-devel/rev/34a31d29f666.

This time we have a relatively short list of NEWS and ANNOUNCE items, but
there are a few heavy things: VSCodium with full Electron/Node.js/Chromium
included, and Scala 3 --- a totally new compiler for a quite different source
language.

Any feedback about Isabelle release candidates should be posted with a meaningful
Subject (not just a clone of the announcement).

Makarius
