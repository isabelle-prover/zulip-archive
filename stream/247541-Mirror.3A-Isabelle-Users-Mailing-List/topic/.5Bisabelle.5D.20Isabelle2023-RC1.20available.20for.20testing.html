Dear Isabelle users,

we can now start of the official release process for Isabelle2023 (August
2023). There are approx. 6-7 weeks left to sort out remaining problems;
afterwards the release will become final and remain unchanged (until the next
release after it).

This time the situation is unusual by having the ITP 2023 conference in the
middle of it: I will be present myself in Białystok.

The release process is continuously documented on the blog entry
https://isabelle-dev.sketis.net/phame/post/view/68/release_candidates_for_isabelle2023

The current release candidate is
https://isabelle.in.tum.de/website-Isabelle2023-RC1

A corresponding version of the Archive of Formal Proofs is
https://isabelle.sketis.net/repos/afp-devel/rev/e2e58e494821.

The NEWS and ANNOUNCE files list key changes. A few notable add-on tools are
still waiting in the pipeline: this material should converge within 2 weeks.

As usual, any feedback about Isabelle release candidates should be posted with
a meaningful
Subject (not just a clone of the announcement).

Makarius
