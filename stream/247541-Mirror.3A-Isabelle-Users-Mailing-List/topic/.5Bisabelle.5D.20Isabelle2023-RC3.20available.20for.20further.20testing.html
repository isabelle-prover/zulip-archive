Dear Isabelle users,

various people have tested earlier release candidates already, so the current
Isabelle2023-RC3 approximates a final release pretty well. See
https://isabelle.in.tum.de/website-Isabelle2023-RC3

Only some updates of documentation is still missing, and a few add-on tools in
the pipeline.

See also
https://isabelle-dev.sketis.net/phame/post/view/68/release_candidates_for_isabelle2023
for further details and last-minute changes.

An important improvement is stable support for Poly/ML on ARM64, e.g. native
Apple Silicon M1/M2: all of Isabelle + AFP works reliably.

As usual, any feedback about Isabelle release candidates should be posted with
a meaningful
Subject --- not just a clone of the announcement.

Makarius
