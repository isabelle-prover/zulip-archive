One notable NEWS entry is:

  • The arm64-linux platform is now officially supported, although a few
    (non-essential) tools are missing:

    • Z3
    • CVC4
    • OCaml / OPAM
    • Nunchaku + smbc (experimental)

It remains to be seen how far we get with that right now:

- "z3" still occurs frequently in AFP entries, but it can be usually
replaced by "verit" --- the Isabelle Z3 component is a bit old and will not be
updated for Isabelle2024.

- CVC4 might be replaced by cvc5 soon, with regular arm64-linux support.

- OCaml is uninportant

- Nunchaku is truly experimental an mostly unused

I do have my own virtual arm64-linux machine now, provided by Netcup.de.
Further note that Docker on Apple Silicon will work with arm64-linux.

Makarius
