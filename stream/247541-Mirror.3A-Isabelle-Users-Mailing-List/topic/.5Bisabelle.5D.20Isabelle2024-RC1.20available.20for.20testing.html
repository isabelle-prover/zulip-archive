Dear Isabelle users,

the official release process for Isabelle2024 (May 2024) starts now, a bit
earlier than the last few releases, but not earlier than usual: the regular
distance between releases has been 8-10 months since 2008. The Isabelle2024
release process is continuously documented on the blog entry
https://isabelle-dev.sketis.net/phame/post/view/76/release_candidates_for_isabelle2024

The current release candidate is
https://isabelle.in.tum.de/website-Isabelle2024-RC1

A corresponding version of the Archive of Formal Proofs is
https://isabelle.sketis.net/repos/afp-devel/rev/12db4a7858c0

The NEWS file is already up-to-date. Some documentation is still coming, in
particular about the improved cluster build tool: that will allow proper
library maintenance once again.

As usual, any feedback about Isabelle release candidates should be posted with
a meaningful Subject (not just a clone of the announcement).

Makarius
