Dear Isabelle users,

we are at the pre-final stage of the release process for Isabelle2025 (March
2025), see also
https://isabelle-dev.sketis.net/phame/post/view/85/release_candidates_for_isabelle2025

The website https://isabelle.in.tum.de/website-Isabelle2025-RC4 provides the
usual downloads for Linux (Intel), Linux (ARM), macOS (Intel and ARM), Windows
(Intel).

Everything is ready for serious applications. Only some documentation updates
are still missing. Rather soon there will be a final and unchangeable version
of Isabelle2025, without any patches until the next release in 8-10 months.

Any feedback about Isabelle release candidates should be posted with a
meaningful Subject, not just a clone of the announcement.
