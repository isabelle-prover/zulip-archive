For this particular version I would like to highlight the new "isabelle
find_facts_server", which should now work properly following this example from
NEWS:

isabelle find_facts_server -p 8080 -o find_facts_database_name=isabelle
open http://localhost:8080/find_facts#search?q=Hilbert

In a later release we will integrate this better into the Prover IDE. Further
note the already existing "Browse HTML Library" menu in Isabelle/jEdit, which
uses the server from the running PIDE JVM. The Find_Facts server uses the same
HTTP service when displaying the HTML library.

Thus it should always show the correct data, and work without a network
connection.
