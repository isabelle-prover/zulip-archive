Hi Angeliki,

You are allowed to import only one session in the initial "session" line, but can import further sessions under the “sessions" heading.

chapter AFP

session Catalan_Numbers = "HOL-Analysis" +
options [timeout = 600]
sessions
Landau_Symbols
"HOL-Real_Asymp"
theories
Catalan_Numbers
document_files
"root.tex"
"root.bib"

Larry
