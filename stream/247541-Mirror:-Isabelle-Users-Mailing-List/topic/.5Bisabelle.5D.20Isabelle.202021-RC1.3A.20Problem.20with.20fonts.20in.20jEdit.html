Starting with Catalina, Apple has moved the platform onto windy islands,
making life much harder for applications.

Luckily, all known GUI problems just disappear with the new FlatLAF
look-and-feel, which is already active by default in Isabelle2021-RC2.

See also https://www.formdev.com/flatlaf --- the current version is numbered
rather modestly as 0.46, but that is already quite well-done and stable.

In contrast, the system look-and-feels of OpenJDK (GTK+, Windows, macOS Aqua)
are more and more falling into decay.

I was briefly considering to switch back to good old "Metal" by default, when
I found this great new project.

Makarius
