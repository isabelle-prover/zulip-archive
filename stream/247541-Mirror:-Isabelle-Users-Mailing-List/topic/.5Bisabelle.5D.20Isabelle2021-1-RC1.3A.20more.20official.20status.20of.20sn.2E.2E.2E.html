There has been a private conversation concerning the lack of the popup in
Isabelle/jEdit to remind of the unofficial status of release candidates.

One reason for its discontinuation is that intermediate snapshots (and
repository clones) have gained a somewhat more official status recently. For
example, this is relevant to "launch" add-on projects that are awaiting a
proper Isabelle release eventually. The Admin/init script within the
repository greatly helps with that, see
https://isabelle.sketis.net/repos/isabelle/file/72d2ef5ee128/Admin/init

It is not in the NEWS, because the communication model of that file is to talk
about proper releases.
