Functional Algorithms, Verified!
http://functional-algorithms-verified.org/

This book is an introduction to data structures and algorithms for functional
languages, with a focus on proofs. It covers both functional correctness and
running time analysis. It does so in a unified manner with inductive proofs
about functional programs and their running time functions.
All proofs have been machine-checked by the proof assistant Isabelle.

Enjoy!
