Thank you all for this work!

Here is a small question/suggestion:

Recently, while helping someone setting up the AFP for Isabelle/HOL on
windows, the instructions for referring to AFP entries in Isabelle/JEdit
for Windows (in https://www.isa-afp.org/help/) did not work. The problem
was fixed when we followed Jakub Kądziołka's suggestion, on Zulip (here:
https://isabelle.zulipchat.com/#narrow/stream/238552-Beginner-Questions/topic/afp/near/240305960)
to append "/thys" to the path to the AFP in the
"~/.isabelle/etc/components" file.

I don't know if this is just a documentation problem to fix on the
website, or if the issue lies with the "isabelle components -u
path/to/afp" command for windows (on linux, I never had a problem with
it), or if my student made a mistake somewhere during the process, but
it could be worth mentioning this issue and its fix in the Help section.

Best,
  Sophie
