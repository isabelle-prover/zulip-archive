We have three new entries, by Achim D. Brucker and Michael Herzberg:

• The Safely Composable DOM (Document Object Model)
https://www.isa-afp.org/entries/Core_SC_DOM.html

• A Formal Model of the Safely Composable Document Object Model with Shadow Roots. Shadow Roots are a recent proposal of the web community to support a component-based development approach for client-side web applications.
https://www.isa-afp.org/entries/Shadow_SC_DOM.html

• A Formalization of Safely Composable Web Components: defines safety properties which ensure that different web components can only interact with each other using well-defined interfaces.
https://www.isa-afp.org/entries/SC_DOM_Components.html
