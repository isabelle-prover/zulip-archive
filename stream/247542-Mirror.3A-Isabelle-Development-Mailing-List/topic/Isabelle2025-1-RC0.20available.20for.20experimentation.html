Dear Isabelle users,

2025 is a year with two Isabelle releases: after Isabelle2025 (March 2025)
there will be Isabelle2025-1 (December 2025). See the blog entry
https://sketis.net/2025/release-candidates-for-isabelle2025-1 for
documentation of the ongoing release process.

Approx. 4 weeks before the first release candidate, there is now an informal
snapshot for early experimentation:
https://isabelle.in.tum.de/website-Isabelle2025-1-RC0

This opens a window of opportunity to provide feedback, until the final
release becomes unchangeable (approx. 15-Dec-2025). As usual, feedback
messages should use a proper Subject line and not just a clone of this
announcement.

Makarius

A few more explanations on the meaning of RC0:

* It is merely a preview of the accidental situation approx. 4 weeks before
RC1, which is the first regular release candidate.

* It is a reminder that the release process is about to start soon.

* It is high time to wrap up, whatever is still pending and relevant for
the release. (I have myself many lose ends to close for RC1/RC2, but some will
probably be postponed to a later release.)

* It is not a good time to come up with new things for a later release.
