  • Nitpick: External solver "MiniSat" is available for all supported Isabelle
    platforms (including Windows and ARM); while "MiniSat_JNI" only
    works for Intel Linux and macOS.

This refers to Isabelle/74a36aae067a and Isabelle/2d089ff0e03b; there is also
an administrative tool isabelle build_minisat.

This solves the longstanding problem to support 64bit Windows. Moreover,
arm64-linux now works as promised for the Isabelle2021-1 release.

One open question: Should Nitpick use "MiniSat" instead of "MiniSat_JNI" by
default (especially in HOL-Nitpick_Examples)?
