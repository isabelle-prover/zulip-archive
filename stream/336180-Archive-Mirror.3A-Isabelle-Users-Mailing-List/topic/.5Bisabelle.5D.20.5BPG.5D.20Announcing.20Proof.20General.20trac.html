Dear Proof General users and developers,

http://proofgeneral.inf.ed.ac.uk/trac

I'm trying to a new way of documenting problems and enhancement
requests by keeping a bug tracker of fixes that we need for Proof
General. Even if the developers don't have time to fix them
themselves, perhaps others will be able to help or suggest
workarounds. It's a frustration that users sometimes complain to one
another about problems and put up with them, but never report them to
the developers.

I don't open source projects can ever expect all users to be bug
reporters, but with a small-ish user community spread across many
different prover systems, operating systems and emacs versions, it
would help if a larger than usual percentage of PG users could be.

This list is small and quiet: please pass the message on to other PG
users.

Many thanks!

- David
