Since the Isabelle repository has forked for the Isabelle2020 release this week, the AFP is following suit as usual.

AFP-2020 is now forked on bitbucket for Isabelle2020-RC2+, and afp-devel continues to refer to the isabelle development version.

If anyone has changes that are important to include in AFP-2020, please commit these to afp-devel and let me know by email before Fri, 27 Mar 2020.

Current AFP submission should still use Isabelle2019. Submissions that work with both versions or need only small changes for Isabelle2020 are appreciated by your editors :-)

Cheers,
Gerwin
