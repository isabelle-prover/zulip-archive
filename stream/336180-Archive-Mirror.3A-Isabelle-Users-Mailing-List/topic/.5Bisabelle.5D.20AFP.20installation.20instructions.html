Hi,

as usually, the problem becomes clear the moment the mail is sent: When
extracting the release
(http://sourceforge.net/projects/afp/files/latest/download) as root it
is not readable for other users (and using Isabelle/jEdit no helpful
error message was shown).

After fixing the permissions with "chmod a+Xr -R afp-2014-11-03/" it
works.

Nevertheless I suggest to unify http://afp.sourceforge.net/using.shtml
and https://isabelle.in.tum.de/community/Building_AFP to use the same
mechanism (if they are indeed equivalent).
