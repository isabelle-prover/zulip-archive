After SF’s recent problems, we've decided to host the AFP repository on Bitbucket instead: https://bitbucket.org/isa-afp/

The SF repository will stay around for a while as a mirror, but will be read-only at this stage.

To (re-)gain write access to the development version of the AFP, please send me your bitbucket account name (sign-up at https://bitbucket.org).

There’s no need to get a new copy of the repository, you can just set the default path in the file .hg/hgrc in the root of your working copy as follows, and continue to work with it as before:

[paths]
default = ssh://hg@bitbucket.org/isa-afp/afp-devel

(This will require setting up an ssh key at bitbucket, you can of course also use the https version)

We’re also planning to move the web site, but this will take a bit more preparation.

Cheers,
Gerwin
