I would like to announce the availability of Babel-17 v0.21.1, a programming language for purely functional programming. It is hosted at www.babel-17.com. There is also a Netbeans plugin for Babel-17, which is currently the featured plugin at netbeans.org .

Why would this be of interest to an Isabelle user? Well, first I think that purely functional programming is something that interests the typical Isabelle user. Second, if somebody is looking for a programming language for which to formulate a semantics in Isabelle, Babel-17 would surely be not too difficult to tackle, being purely functional and all :-) A slight complication might be the linear scope rules of Babel-17.
