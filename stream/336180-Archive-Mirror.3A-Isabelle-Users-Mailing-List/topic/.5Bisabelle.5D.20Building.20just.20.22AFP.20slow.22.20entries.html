is it at all possible to run just the AFP entries tagged "slow"? If I
use "isabelle build" with the flags "-d $AFP -g slow" it also builds

Email Gateway (Aug 22 2022 at 11:58):

From: Makarius <makarius@sketis.net>
HOL-Proofs is also tagged as "slow". Options -x and -X allow to exclude
sessions from the selection, e.g. like this:

isabelle build -d '$AFP' -n -v -g slow -x HOL-Proofs

In more complex situations, Isabelle/Scala may be used to specify session
build jobs under program control.

Makarius
