From: Mark A Hills <mhills@cs.uiuc.edu>
This is because impl is invalid -- you are intending to use impI
instead. So, this would work:

theory simple=HOL:
lemma first_theorem: "A-->(B-->A)"
apply(rule impI)
apply(rule impI)
apply assumption
done
end
