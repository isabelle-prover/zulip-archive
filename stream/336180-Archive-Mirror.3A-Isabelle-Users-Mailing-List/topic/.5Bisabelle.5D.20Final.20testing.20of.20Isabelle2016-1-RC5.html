Dear Isabelle users,

the final release of Isabelle2016-1 is scheduled for next week. There is
one last release candidate:
http://isabelle.in.tum.de/website-Isabelle2016-1-RC5 with some important
changes:

1. more uniform automatic indentation (empty vs. non-empty lines)
2. fewer tracing/warning messages in some proof tools

The main difference of Isabelle2016-1-RC5 to the anticipated
Isabelle2016-1 release is that the final release will be final. Any
remaining problems can be only addressed now, while we are still testing.

AFP is already forked for the release, see
https://bitbucket.org/isa-afp/afp-2016-1

Makarius
