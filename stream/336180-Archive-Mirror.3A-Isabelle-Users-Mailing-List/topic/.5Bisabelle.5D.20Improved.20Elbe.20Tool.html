Good Morning,
there is an update of the Elbe software for Windows, which lacks several
bugs of the earlier version. [Elbe is a language tool for navigation and
pretty printing of source files and a simple batch runner.] I corrected
the Isabelle.ttf font and called it Elbian.ttf and with that font the
Acrobat export now works. Also two languages have been added: Fortran and
VHDL. The main improvement is a navigator, which makes the language files
easier to investigate, so that one can see at once, what is contained in a
source like e.g. rationals.thy. Those who are interested can download it
from
http://cococo.de/Context_IT_GmbH/index.jsp?content=elbe
Have a successful week,
Jens
