Is this really Windows 8, and not 8.1? Microsoft has provided a free
update long ago, because Windows 8 proper had various problems. (Anyway,
there is no particular reason why Windows 8 should not work here.)

Can you check the state of the above-mentioned file with the Windows
explorer or some other tool? The location is in symbolic Isabelle POSIX
format. You can get the physical windows path like this:

* Execute Isabelle2015\Cygwin-Terminal.bat by double-clicking it

* In the terminal window run this command:

isabelle getenv -b ISABELLE_HOME_USER

or even this:

cygpath -w $(isabelle getenv -b ISABELLE_HOME_USER)

The latter form provides a literal Windows path, with drive letters and
backslashes.

There might be actually something wrong with this $ISABELLE_HOME_USER
directory, and not the etc/preferences file itself. If the file is there
nonetheless, but looks odd, you may just delete it an try again.
