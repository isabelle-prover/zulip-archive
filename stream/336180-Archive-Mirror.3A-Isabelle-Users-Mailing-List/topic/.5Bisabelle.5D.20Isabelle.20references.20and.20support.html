I don't think there is anything special about mathematical applications of
Isabelle. Everything is ultimately mathematics, even if
computer-scientists sometimes pretend that it is all about computation and
program verification.

So you can just look at the regular examples and documentation of
Isabelle. E.g. see the Documentation panel in the Prover IDE.

It might help to start clicking through existing theories and look a bit
at the screenshots with explanations in the Isabelle/jEdit manual.
