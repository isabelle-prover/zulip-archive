Dear Isabelle users,

this is presumably the last release candidate before the Isabelle2014
release next week. See:

http://isabelle.in.tum.de/website-Isabelle2014-RC4

Notable changes to Isabelle2014-RC3:

* Update to latest jdk-7u67

Oracle says: "This release addresses a regression introduced in the
most recent security update."

* Update to recent Cygwin 1.7.31-3
Corinna Vinschen says: "This is another bugfix release to fix the
bugfix release."

* More convenient reports on low-level Isabelle/PIDE failures via Syslog
panel, instead of awkward Raw Output panel.

* More robust treatment of removed PIDE document versions (after 3 years
of routine use).

Observations and problems of release candidates may be discussed here on
isabelle-users (using an informative subject for the mail thread), or via
private mail to the person who is responsible (when that is obvious).

The final release is will be published next week, before the end of the
month. It will be really final: problems that are not reported now are
doomed to remain. This is the very last chance to participate in testing,
and thus improving the overall quality of the release.

Makarius
