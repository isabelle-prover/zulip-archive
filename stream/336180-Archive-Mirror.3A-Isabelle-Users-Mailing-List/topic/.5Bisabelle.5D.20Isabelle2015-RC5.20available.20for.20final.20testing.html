Dear Isabelle users,

Isabelle2015-RC5 is now available:

http://isabelle.in.tum.de/website-Isabelle2015-RC5

Changes can be inspected formally here:

https://bitbucket.org/isabelle_project/isabelle-release/commits

Here is an informal summary of notable points:

- avoid crash of print_simpset
- clarified name space accesses for local entities within "context begin ... end"
- further tuning of Eisbach implementation
- added Eisbach User Manual

People who have tested earlier release candidates should upgrade now.
Local settings can be preserved by copying $ISABELLE_HOME_USER in a
suitable manner: renaming the last component of the directory name to
Isabelle2015-RC5 before executing the new version.

This is the last release candidate, before Isabelle2015 is finalized,
presumably in 1 week. Any remaining and unreported problems of
Isabelle2015-RC5 will thus become final as well!

Makarius
