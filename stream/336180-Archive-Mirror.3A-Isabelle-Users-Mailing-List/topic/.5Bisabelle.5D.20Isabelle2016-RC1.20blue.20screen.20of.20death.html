A BSOD is definitely bad, but usually in the responsiblity of some big

The main difference of Isabelle2016-RC1 on win-32 vs. win-64 is the Java
Runtime Environment, and its default heap parameters. You may change the
latter in the file Isabelle2016-RC1.l4j -- these are standard java options
as explained here
http://docs.oracle.com/javase/8/docs/technotes/tools/windows/java.html
