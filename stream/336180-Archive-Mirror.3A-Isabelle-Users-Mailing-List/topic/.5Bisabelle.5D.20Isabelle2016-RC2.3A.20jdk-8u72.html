A few more notes on this. Oracle has delivered its scheduled Java 8
update on time, and it looks generally quite good. More details are here:
http://www.oracle.com/technetwork/java/javase/2col/8u72-bugfixes-2775805.html

For example, the broken Nimbus scrollbar "thumb" works again (see also
https://bugs.openjdk.java.net/browse/JDK-8134828). Nonetheless, the
default look-and-feel on Linux remains Metal, since that can be more
easily used with very high resolution displays ("UHD" or "4K" or more).

There is always a hope that other platform specific problems have now
disappeared, but it needs proper testing, especially on Mac OS X.

I've also removed overly ambitious JVM GC options: the situation is back
to Isabelle2015 in that respect. This might impact system stability, e.g.
on Windows.
