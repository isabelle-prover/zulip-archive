Dear Isabelle users,

Isabelle2016-RC4 is now available for further testing, and regular use in
anticipation of official Isabelle2016:

http://isabelle.in.tum.de/website-Isabelle2016-RC4

This corresponds to the repository versions
https://bitbucket.org/isabelle_project/isabelle-release/commits/Isabelle2016-RC4
and https://bitbucket.org/isa-afp/afp-devel/commits/100836987bd3

Summary of notable points:

- more robust Sledgehammer preplay phase
- more reactive GUI updates
- minor tuning and polishing

After 5 weeks of testing, extremely few problems have been exposed. This
means we are converging to Isabelle2016 -- this is probably the last
release candidate before final lift-off.

People who have used earlier release candidates should upgrade now. Local
settings can be preserved by copying $ISABELLE_HOME_USER in a suitable
manner: renaming the last component of the directory name to
Isabelle2016-RC4 before starting the new version.

When discussing problems, observations, suggestions, etc. the mail subject line
should be changed to something informative (but the release candidate number
still given in the message body).

Makarius
