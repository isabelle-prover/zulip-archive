Dear Isabelle users,

the coming Isabelle2017 release is scheduled for October 2017. In the
coming weeks we have plenty of time for thorough testing of release
candidates, presently http://isabelle.in.tum.de/website-Isabelle2017-RC2

The corresponding repository versions of Isabelle and AFP are
https://bitbucket.org/isabelle_project/isabelle-release/commits/e9d8ff531700
and
https://bitbucket.org/isa-afp/afp-devel/commits/dfb6f8bc70e3

The website, NEWS, ANNOUNCE etc. are already up-to-date, but some
documentation still requires update.

See also http://sketis.net/2017/release-candidates-for-isabelle2017 for
notable changes within the line of release candidate.

It is important to note that snapshots from
http://isabelle.in.tum.de/devel already follow the development process
after the Isabelle2017 release. From now on, it only makes sense to
test proper release candidates.

When posting problems, observations, suggestions, etc. the mail
subject line should be changed to something meaningful, and the
particular Isabelle version given in the message body.

Makarius
