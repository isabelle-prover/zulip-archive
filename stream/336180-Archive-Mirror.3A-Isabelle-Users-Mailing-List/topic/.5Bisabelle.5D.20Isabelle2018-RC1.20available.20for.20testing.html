Dear Isabelle users,

the coming Isabelle2018 release is scheduled for August 2018, approx. 6
weeks from now. The first official release candidate is now available
for general testing: https://isabelle.in.tum.de/website-Isabelle2018-RC1

The corresponding repository versions of Isabelle and AFP are
https://bitbucket.org/isabelle_project/isabelle-release/commits/cf01d04e94d7
and https://bitbucket.org/isa-afp/afp-devel/commits/2af750da996c

The website, NEWS, ANNOUNCE etc. are already up-to-date, but some
documentation still requires update.

The ongoing release process is continuously documented on my blog:
https://sketis.net/2018/release-candidates-for-isabelle2018

When discussing observations about release candidates, please provide
a mail Subject line that fits to the content, not just a clone of the
announcement.

The time for testing and reporting problems is now! After the release
has been finalized and published, it is too late to change anything.

Makarius
