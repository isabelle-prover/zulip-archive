You have rediscovered a problem of current OpenJDK 11.0.6 that was already
marked as "resolved" and "backported" here:

https://bugs.openjdk.java.net/browse/JDK-8217731
https://bugs.openjdk.java.net/browse/JDK-8230316

The same happens for latest Adopt OpenJDK 14+36.

What is odd about it: earlier versions before the "fix" don't have it. So we
are now back to Adopt OpenJDK 11.0.5:

https://isabelle.sketis.net/repos/isabelle-release/rev/0f98a7c366ed

There is formally no need to be on "latest" versions, unless there is really
something wrong with older ones.

We are only missing many small changes according
https://adoptopenjdk.net/release_notes.html?variant=openjdk11&jvmVariant=hotspot
