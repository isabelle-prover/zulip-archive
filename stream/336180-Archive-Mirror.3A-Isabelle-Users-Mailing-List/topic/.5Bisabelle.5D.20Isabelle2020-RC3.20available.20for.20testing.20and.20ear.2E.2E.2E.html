Dear Isabelle users,

an even more consolidated release candidate for Isabelle2020 (April 2020) is now
available from https://isabelle.in.tum.de/website-Isabelle2020-RC3

See again the blog entry
https://isabelle-dev.sketis.net/phame/post/view/5/release_candidates_for_isabelle2020
for details on various small changes that have accumulated so far.

Recall that the Isabelle development process is very efficient and light
weight: instead of continous socialising concerning "bugs" and "fixes" there
is only a brief period before the final release to sort things out. Afterwards
the release remains final and unchangeable for 8-10 months.

It means there are only 3 weeks left until Isabelle2020 becomes immutable.

When discussing observations about Isabelle2020 release candidates on the
mailing list, please provide a "Subject:" line that fits to the content, not
just a clone of this announcement.

Makarius
