Hello all,

now there is more functionality in the Elbe IDE
(http://cococo.de/Elbe/). The features are

* full authoring of Isabelle proofs at symbol level
* mathematical Font included
* now JAVA, Bash, C++, XML compatible
* Linux and Windows files, (\n or \r\n line endings)
* source navigation by method/procedure/function/lemma
* beautiful print preview
* data items navigation for Fortran and Cobol
* programming aid by keyword/data item/label/lemma name
* batch processing
* regular expressions
* faster load time
...

Please have a look at the product and tell me if you want some/other
features added.
