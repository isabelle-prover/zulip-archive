FOM mailing list doesn't have this problem, so configuring mailman is possible. If you don't see config option, try to edit HTML templates or CSS files (if they are available). For example, you can apply the following patch to https://lists.cam.ac.uk/camstyle.css :

-pre, tt {font-family: Courier, monospace; font-size: 90%;}
+pre, tt {font-family: Courier, monospace; font-size: 90%; white-space: pre-wrap;}
