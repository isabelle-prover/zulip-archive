The meta theory of the Incredible Proof Machine
Joachim Breitner and Denis Lohner

The Incredible Proof Machine is an interactive visual theorem prover which
represents proofs as port graphs. We model this proof representation in
Isabelle, and prove that it is just as powerful as natural deduction.

Check out the Incredible Proof Machine http://incredible.pm/
and come to ITP in Nancy to see the presentation of this work.
