Funding for a PhD project on verification of efficient algorithms (in
Isabelle/HOL) is available at the University of Manchester.

Please disseminate to potentially interested students.

Short Description: This project aims at extending the current LLVM code
generator for Isabelle/HOL, and developing new tools and techniques for
the verification of efficient algorithms, down to their implementation
in LLVM.
