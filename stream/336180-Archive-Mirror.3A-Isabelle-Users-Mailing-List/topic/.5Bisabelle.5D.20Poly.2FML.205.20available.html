Dear Isabelle users,

David Matthews has announced the official release of Poly/ML 5, see
http://www.polyml.org/ and
http://www.polyml.org/docs/Version5ReleaseNotes.html in particular. This
version supports various additional platforms, notably Intel Macs and
AMD64 machines.

Those who would like to give it a try with Isabelle2005 may get the
required compatibility files and some README from
http://www4.in.tum.de/~wenzelm/test/Isabelle2005-polyml-5.0.tar.gz
Note that you will have to compile Isabelle logics manually.

Makarius
