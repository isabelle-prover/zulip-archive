Dear Isabelle users,

this is just a straw poll if anyone still remembers or even uses DVI files
for Isabelle document preparation -- this includes the generated Isabelle
documentation https://isabelle.in.tum.de/documentation.html, although that
WWW version is in PDF as usual.

I used to be myself a user of DVI until some years ago, when the extremely
fat and slow Adobe reader was eventually replaced by derivatives of
Poppler (e.g. evince) on Linux. The alternative PDF reader that is
bundled with Isabelle2013 on Windows is also very small and fast, and Mac
OS X had smooth support for PDF from the start.

So it looks like the time to dismantle alternative DVI (and PS) output
could be now, unless someone points out convincing applications of it.

Makarius
