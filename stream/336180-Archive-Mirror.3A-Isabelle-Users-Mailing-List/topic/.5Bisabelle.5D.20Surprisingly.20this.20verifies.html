Two more remarks:

theory test
imports HOL Main

There is no need to import HOL. In fact, you shouldn't. Simply
importing Main will be sufficient.

definition "f (t1, x) = t1"
definition "f (t1, x) = t2"

These definitions are not contradictory; they merely imply t1 = t2.
