When I mix theory files with LaTeX markup, the size of the font of the
Isabelle output (from, have, by etc.) changes with the size of the
surrounding text. Is there a way to fix the size of the Isabelle font
regardless of what size the surrounding fonts are? I think it should
be possible by adding:

\renewcommand{Somthing}{10}

in root.tex, but I don't know what the Something should be!

Many thanks

Peter

Email Gateway (Aug 18 2022 at 13:58):

From: Makarius <makarius@sketis.net>
The LaTeX style for the formal parts of the text are essentially hardwired
to what \isastyle, \isastyleminor, \isastylescript, which can be redefined
as usual. See isabelle.sty for details.
