Hi Eg,

datatype graph = "('a, 'b)"
using this you have defined a datatype with one singleton constructor
named "('a, 'b)". What you had in mind was something like

datatype ('a, 'b) graph = Graph 'a 'b

which is already present with the product type.

Hope this helps,
