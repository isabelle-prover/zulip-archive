Sam TH wrote:
You're most likely using an older version of ProofGeneral which is
incompatible with the development version of Isabelle. To solve the
problem, you could try to use the CVS version of ProofGeneral
(see http://proofgeneral.inf.ed.ac.uk/devel) or try the pre-release
version. If you can't get the "Settings" submenu to work, you can
also switch on "Show Types" by issuing the command

ML "set show_types"
