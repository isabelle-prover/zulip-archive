Sorry, this does not work. All these components are fit together tightly,
and cannot be exchanged easily with other versions found in the wild.

Especially for Isabelle/jEdit I've made some fine tuning, with some
properties and a few lines of patches for the main code base, apart from
adding certain plugin and non-plugin jars in the right spot. Moreover,
the user settings directory is specific for each Isabelle version, to keep
things disjoint and thus working independently of other jEdits and
Isabelle versions.

If you want a newer jEdit right now, you should try Isabelle2012-RC1. It
is based on the latest stable jEdit release.

Makarius
