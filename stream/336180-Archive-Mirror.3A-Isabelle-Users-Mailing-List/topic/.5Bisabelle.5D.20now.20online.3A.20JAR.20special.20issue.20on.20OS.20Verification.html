We are pleased to announce that the JAR special issue on OS Verification
has now appeared on SpringerLink:

http://springerlink.com/content/r17308r375hw/?p=b05507b4fe8f4cc19d73c0c4d22aaf19&pi=1

The special issue contains seven articles on theorem proving, model
checking, and static analysis from the leading research groups in OS
verification. The articles range from formal modelling of C code and virtual
memory, over specific OS formalisations to formal models for low-level
concurrency and model checking techniques in the OS verification area.
