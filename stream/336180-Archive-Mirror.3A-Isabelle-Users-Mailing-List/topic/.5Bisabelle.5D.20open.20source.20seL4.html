General Dynamics and NICTA are pleased to announce that the formally verified seL4 microkernel and its proofs will be released as open source on 29th July 2014, the 5th anniversary of the first completion of its functional correctness proof.

C code and Isabelle proofs are planned to be hosted on github. The web site will be populated with more information as we progress towards the release date.
