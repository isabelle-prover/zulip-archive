I want to switch my "Isabelle" to startup gnu emacs instead of xemacs.

Email Gateway (Aug 18 2022 at 10:46):

From: Makarius <makarius@sketis.net>
Running the Isabelle interface script with option -? will give you the
usage of that script (which happens to be part of ProofGeneral). You may
then add options to your liking via PROOFGENERAL_OPTIONS in
~/isabelle/etc/settings, e.g. something like "-p emacs" to specify plain
GNU Emacs (or any more specific path to an emacs executable).

Makarius
