From: Florian Haftmann <florian.haftmann@informatik.tu-muenchen.de>
Hi Ruben,

you can change the default emacs by tweaking the etc/settings file in
your Isabelle Distribution, e.g.:

PROOFGENERAL_OPTIONS="-p xemacs"

Note that ProofGeneral is very selective in emacs versions it works
properly with. Usually the default settings are quite ok. But I
personally am not a Mac user and cannot tell you anything particular
about which emacs version works best on Mac. Anyway
http://isabelle.in.tum.de/installation.html recommends XEmacs 21.4. for
MacOS.

Hope this helps
